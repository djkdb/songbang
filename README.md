# 🎤 송방 SongBang — 노래방 랜덤 뽑기

> 노래방에서 "뭐 부르지?" 하며 인기차트를 한참 뒤지는 시간, 이제 끝!
> 내가 부를 수 있는 노래를 저장해두고, 버튼 한 번으로 랜덤으로 뽑아주는 노래방 도우미 앱입니다.

## ✨ 주요 기능

### 🎰 랜덤 뽑기
- **내 노래 / 인기차트 / 둘 다** 중에서 뽑기 소스 선택
- 장르 필터 (발라드, 댄스, 락/밴드, 힙합, R&B, 트로트, OST, 포크/인디, 팝)
- **오늘 이미 뽑은 곡 제외** 옵션 — 같은 곡이 또 나오는 것 방지
- 슬롯머신 애니메이션으로 두구두구 🥁 하는 재미
- 최근 뽑은 노래 기록 (최대 30곡)

### 🎤 내 노래 목록
- 제목 / 가수 / **TJ·금영 노래방 번호** / 태그 저장
- 검색, 수정, 삭제
- JSON 파일로 **내보내기·가져오기** (기기 옮길 때 유용)
- 모든 데이터는 브라우저 localStorage에 저장 — 서버 없이 동작

### 🔥 인기차트 100
- 최신 히트곡 + 노래방 스테디셀러를 섞은 내장 차트
- 제목·가수 검색, 장르 필터
- ♥ 버튼 한 번으로 내 노래에 담기
- 차트에서 뽑힌 곡도 바로 내 노래에 추가 가능

### 📱 PWA (앱처럼 설치)
- 홈 화면에 추가하면 앱처럼 실행
- 서비스 워커로 **오프라인에서도 동작** — 지하 노래방에서 신호가 안 터져도 OK

## 🚀 실행 방법

빌드 과정이 없는 순수 HTML/CSS/JS 앱이에요.

```bash
# 아무 정적 서버로 열면 됩니다
npx serve .
# 또는
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속. (`index.html`을 직접 열어도 동작하지만, PWA/서비스 워커는 http(s) 환경에서만 활성화됩니다.)

### GitHub Pages 배포
저장소 설정 → Pages → 브랜치 선택만 하면 바로 배포됩니다. 별도 빌드 설정이 필요 없어요.

## 📁 프로젝트 구조

```
songbang/
├── index.html            # 앱 진입점 (뽑기/내 노래/인기차트 탭)
├── css/style.css         # 네온 노래방 테마 스타일
├── js/
│   ├── app.js            # 앱 로직 (뽑기, 목록 관리, 저장)
│   └── chart-data.js     # 내장 인기차트 100 데이터
├── icons/                # 앱 아이콘 (SVG)
├── manifest.webmanifest  # PWA 매니페스트
└── sw.js                 # 오프라인 캐시 서비스 워커
```

## 📝 참고

- 인기차트는 앱에 내장된 정적 데이터입니다. `js/chart-data.js`를 수정하면 차트를 갱신할 수 있어요.
- 노래방 기기 번호(TJ/금영)는 기기 버전마다 달라서 차트에는 넣지 않았어요. 내 노래에 담은 뒤 직접 적어두면 다음부터 바로 보여요.
